# Sleep Calculator / Schlafzyklus-Rechner (Wordpress Plugin)

## Beschreibung
Der Schlafrechner hilft dir, deine optimale Schlafzeit basierend auf natürlichen 90-Minuten-Schlafzyklen zu berechnen.

## Funktionen
<ul>
<li>Aufwachzeit berechnen: Gib dein Alter und deine Einschlafzeit ein</li>
<li>Schlafenszeit berechnen: Gib dein Alter und deine gewünschte Aufwachzeit ein</li>
<li>Altersbasierte Empfehlungen (nach WHO-Richtlinien)</li>
<li>Zeigt 4 Optionen (3–6 Schlafzyklen) mit Ideal-Markierung</li>
<li>Optional: Gewünschte Einschlafzeit einrechnen</li>
</ul>

## Shortcode
<code>[schlafrechner]</code>
