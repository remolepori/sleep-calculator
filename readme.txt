Berechne deine optimale Aufwach- oder Schlafenszeit basierend auf Schlafzyklen.

== Description ==
Der Schlafrechner hilft dir, deine optimale Schlafzeit basierend auf natürlichen 90-Minuten-Schlafzyklen zu berechnen.

**Funktionen:**
* Aufwachzeit berechnen: Gib dein Alter und deine Einschlafzeit ein
* Schlafenszeit berechnen: Gib dein Alter und deine gewünschte Aufwachzeit ein
* Altersbasierte Empfehlungen (nach WHO-Richtlinien)
* Zeigt 4 Optionen (3–6 Schlafzyklen) mit Ideal-Markierung

**Einbindung:**
Füge den Shortcode `[schlafrechner]` auf einer beliebigen Seite oder in einem Beitrag ein.

== Installation ==
1. Lade den Plugin-Ordner `schlafrechner` in das Verzeichnis `/wp-content/plugins/` hoch.
2. Aktiviere das Plugin im WordPress-Backend unter „Plugins".
3. Füge den Shortcode `[schlafrechner]` auf einer Seite ein.
